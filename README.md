\# Media Collection Manager



A C++17 media management application that manages books, movies, and television shows.



\## Technologies



\- C++17

\- STL

\- Lambdas

\- File I/O

\- Dynamic Memory Management



\## Concepts Demonstrated



\- Object-Oriented Programming

\- Inheritance

\- Polymorphism

\- Observer Pattern

\- Factory Method Pattern

\- Resource Management
